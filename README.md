## 🛑 Managing the Kill Switch

The kill switch protects your data if your VPN connection drops. It stops all internet traffic until the secure connection returns.

1.  Open the Settings menu inside the application.
2.  Select the Network tab.
3.  Locate the Kill Switch toggle.
4.  Click the switch to turn it on.
5.  The system now monitors your connection constantly.
6.  If the connection fails, the application cuts off traffic to prevent accidental data leaks.

## ✂️ Configuring Split Tunneling

Split tunneling lets you choose which apps use the VPN and which apps bypass it. This helps if you need to access local network devices while staying secure.

1.  Go to the Settings menu.
2.  Select the Split Tunneling section.
3.  Choose the Enable Split Tunneling option.
4.  View the list of installed applications on your computer.
5.  Select the apps you want to send through the VPN tunnel.
6.  Select the apps you want to bypass the VPN.
7.  Save your configuration to apply the changes to your system.
